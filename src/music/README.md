## Features

- **Loop**: Music will repeat automatically
- **Volume**: Set to 30% by default
- **Browser Compatibility**: Works on all modern browsers
- **Auto-play Prevention**: User must click to play (browser requirement)

## Troubleshooting

If music doesn't play:
1. Check that `beautiful-in-white.mp3` exists in `public/music/`
2. Ensure the file is a valid MP3
3. Check browser console for any errors
4. Make sure you're using HTTPS (required for audio in some browsers)
